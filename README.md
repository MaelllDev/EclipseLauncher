# Eclipse Launcher

A simple Minecraft Launcher project built with **C#**, **.NET**, and **CmlLib.Core**.

This repository is mainly for learning and documenting the development process while building a launcher from scratch.

> **Work in progress.** Nothing here is finished yet.

## Current Goals

* Learn how CmlLib.Core works
* Launch Minecraft
* Download game versions
* Understand how launchers are built

## Technologies

* C#
* .NET
* CmlLib.Core
